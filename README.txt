Use Notes:

1. Click on the .exe file which will open the command prompt

2. Paste in the path of the file you want to turn into a .exe

3. Hit enter

4. The file should be created in the same folder as the input file
    A print line will be made in the command line giving confirmation
    Or other messages will indicate an error with the process

Dev Notes:

Remember the files have a preamble which is processed by the first function
and the data itself is handled by the second
